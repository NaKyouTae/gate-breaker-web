# 채팅방 슬래시 명령어

채널 채팅방에서 `/` 로 시작하는 명령어를 입력하여 다양한 액션을 수행할 수 있습니다.

## 명령어 목록

| 명령어 | 설명 | 사용법 |
|--------|------|--------|
| `/강화` | 무기 강화를 시도합니다 | `/강화` 입력 후 무기 선택 |
| `/던전입장` | 채널 멤버를 던전에 초대합니다 | `/던전입장` 입력 후 던전 선택 |

---

## /강화

채팅방에서 바로 무기 강화를 시도할 수 있습니다.

### 사용 흐름
1. 채팅 입력란에 `/강화` 입력
2. 강화 가능한 무기 목록이 표시됨
3. 강화할 무기를 선택
4. 강화 이펙트가 화면에 표시됨 (성공: 금색 폭발 / 실패: 붉은 균열)
5. 결과가 시스템 메시지로 채팅에 표시됨

### 채팅 결과 메시지 예시
- 성공: `@나규태 【 강화 성공 +12 → +13】`
- 실패: `@나규태 【 강화 실패 +12 유지】`
- 파괴: `@나규태 【 강화 실패 — 아이템 파괴됨】`

### 표시 정보
- 사용 골드
- 보유 골드
- 획득 아이템 (강화 레벨 포함)

---

## /던전입장

채널 멤버들과 함께 던전에 입장할 수 있습니다.

### 사용 흐름
1. 채팅 입력란에 `/던전입장` 입력
2. 입장 가능한 던전 목록이 표시됨
3. 던전을 선택하면 채널에 초대 메시지가 전송됨
4. 다른 멤버들이 참여 버튼으로 합류 가능
5. 호스트가 시작 버튼을 누르면 전원 입장

### 제한 사항
- 던전별 최소 레벨 제한 있음
- 채널당 하나의 던전 초대만 활성 가능
- 호스트만 던전 시작/취소 가능
