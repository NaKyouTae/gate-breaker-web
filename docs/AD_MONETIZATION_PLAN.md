# Gate Breaker 광고 기획 문서 (초안)

## 1. 목적
- 게임 플레이 몰입을 해치지 않으면서 광고 수익 구조를 도입한다.
- 광고는 `강제 노출 최소화 + 선택형 보상 강화`를 기본 원칙으로 운영한다.

## 2. 광고 타입별 운영 방향

### 2.1 배너 광고
- 정책: 각 화면에 배너 광고를 배치한다.
- 기본 위치: 하단 네비게이션과 겹치지 않는 화면 본문 하단 영역.
- 공통 규칙:
  - 전투 중 핵심 조작 영역을 가리지 않는다.
  - 채팅 입력창, 버튼 근접 영역과 분리한다.
  - Safe Area(노치/홈 인디케이터)를 고려한다.

### 2.2 네이티브 광고
- 적용 화면: `상점`, `도감`
- 목표: UI 일체감을 유지하면서 광고 피로도를 낮춘다.
- 디자인 원칙:
  - 카드형 UI로 기존 리스트 카드와 동일한 구조/간격 사용
  - `광고` 라벨을 명확히 표시
  - 일반 콘텐츠와 구분 가능한 최소한의 스타일 차이(테두리, 배경 톤) 적용

### 2.3 보상형 광고
- 적용 정책: `일일 던전 클리어 횟수 제한`을 초과한 경우, 광고 시청으로 추가 플레이를 제공한다.

## 3. 화면별 광고 배치안

| 화면 | 배너 | 네이티브 | 비고 |
|---|---|---|---|
| 프로필 | O | X | 하단 정보 카드 아래 |
| 던전 목록 | O | X | 진행 중 던전 플로팅과 겹치지 않게 배치 |
| 인벤토리 | O | X | 장비 상세 모달 열림 시 배너 비노출 권장 |
| 상점 | O | O | 상품 카드 사이 네이티브 슬롯 삽입 |
| 채널 | O | X | 입력창/메시지 영역과 분리 |
| 도감 | O | O | 카드 그리드 중간/하단에 네이티브 삽입 |
| 전투 | X(권장) | X | 몰입 유지 우선, 전투 중 광고 비노출 |

## 4. 일일 던전 클리어 + 보상형 광고 설계

### 4.1 기본 정책
- 일일 기본 클리어 가능 횟수: `N회` (운영 변수)
- N회 초과 시:
  - 광고 미시청: 던전 입장 제한
  - 광고 시청: 추가 클리어 기회 `+1회` 지급

### 4.2 권장 초기값 (운영안)
- `dailyBaseClearLimit = 10`
- `dailyAdUnlockLimit = 5` (광고로 해금 가능한 최대 추가 횟수)
- 하루 최대 클리어: `10 + 5 = 15회`

### 4.3 UX 플로우
1. 유저가 던전 입장 시도
2. 일일 기본 횟수 초과 여부 확인
3. 초과 상태면 팝업 노출:
   - 제목: `일일 클리어 횟수를 모두 사용했습니다`
   - 안내: `광고를 시청하면 추가로 1회 입장할 수 있습니다`
   - 버튼: `광고 보고 입장`, `취소`
4. 광고 시청 완료 시 즉시 입장 허용

### 4.4 실패/예외 처리
- 광고 로드 실패: 재시도 버튼 제공
- 광고 중단/미완료: 추가 횟수 미지급
- 일일 추가 횟수도 소진 시: 다음 리셋 시간 안내

## 5. 운영 파라미터
- `dailyBaseClearLimit`: 일일 기본 클리어 횟수
- `dailyAdUnlockLimit`: 광고로 해금 가능한 일일 추가 횟수
- `adCooldownSeconds`: 광고 시청 재요청 최소 간격
- `dailyResetTimezone`: 일일 리셋 기준 시간대 (예: Asia/Seoul)

## 6. KPI 및 모니터링
- 배너:
  - 노출수(IMP), 클릭률(CTR), eCPM
- 네이티브:
  - 노출 위치별 CTR, 스크롤 도달률
- 보상형:
  - 광고 시청 완료율
  - 광고 1회당 추가 클리어 전환율
  - 광고 유저/비광고 유저 잔존율 차이

## 7. 밸런스/UX 가드레일
- 전투 중 강제 광고 금지
- 광고 시청 유도는 가능하되, 기본 플레이는 항상 유지
- 광고 보상은 `편의/추가 기회` 중심으로 제한 (과도한 P2W 방지)
- 초기 런칭 시 빈도 보수적으로 시작 후 데이터 기반 조정

## 8. 단계별 적용 순서
1. 배너 광고 전 화면 배치 (전투 제외 권장)
2. 상점/도감 네이티브 광고 슬롯 적용
3. 일일 던전 클리어 제한 + 보상형 광고 해금 로직 적용
4. KPI 확인 후 일일 제한/광고 해금 횟수 튜닝
