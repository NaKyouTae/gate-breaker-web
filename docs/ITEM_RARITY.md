# 아이템 등급 컬러 시스템

무기/아이템의 등급(Rarity)은 별도의 텍스트 뱃지 없이 **아이템 이름의 색상**으로만 구분합니다.

## 등급별 색상

| 등급 | 코드 | 색상(HEX) | 용도 |
|------|------|-----------|------|
| COMMON | 일반 | `#888888` | 회색 - 기본 무기 |
| RARE | 레어 | `#4a9eff` | 파란색 - 희귀 무기 |
| EPIC | 에픽 | `#b048f8` | 보라색 - 영웅 무기 |
| LEGENDARY | 전설 | `#ff8c00` | 주황색 - 전설 무기 |
| MYTHIC | 신화 | `#ff2d55` | 빨간색 - 신화 무기 |

## 등급별 글로우(Glow) 색상

강화 화면 등 강조가 필요한 곳에서 사용합니다.

| 등급 | Glow 색상 |
|------|-----------|
| COMMON | `rgba(136,136,136,0.3)` |
| RARE | `rgba(74,158,255,0.4)` |
| EPIC | `rgba(176,72,248,0.4)` |
| LEGENDARY | `rgba(255,140,0,0.5)` |
| MYTHIC | `rgba(255,45,85,0.5)` |

## 적용 위치

- **인벤토리**: 무기 탭 아이템명, 모달 타이틀
- **상점**: 아이템명, 구매 확인 모달 아이템명
- **강화**: 리스트 아이템명, 강화 상세 아이템명 + 글로우
- **전투 결과**: 드롭 아이템명

## 규칙

- 등급 텍스트(COMMON, RARE 등)를 Badge로 표시하지 않습니다
- 아이템 이름 자체의 `color` 속성에 등급 색상을 적용합니다
- 무기는 ATK 스탯만 표시합니다 (HP/DEF는 캐릭터 레벨업으로 성장)
- 강화 레벨은 아이템명 뒤에 `(+N)` 형태로 같은 색상으로 표시합니다
